# مُطابق — لوحة قيادة الامتثال الموحّد

منصّة سعودية تراقب امتثال المنشآت عبر ست جهات (نطاقات/السعودة، الزكاة والدخل،
الفاتورة الإلكترونية «زاتكا»، التأمينات «GOSI»، السجل التجاري، والرخصة البلدية «بلدي»)،
وتنبّه المنشأة استباقياً قبل الغرامة عبر «حارس مُطابق».

## الملفّات
- `index.html` — لوحة القيادة (الكوكبِت) — محميّة بالدخول وقارئة من Supabase.
- `auth.html` — تسجيل الدخول / إنشاء حساب / استعادة كلمة المرور.
- `manage.html` — إدارة المنشآت ومحرّر حالة الجهات الستّ.
- `plans.html` — الباقات والاشتراك (Moyasar).
- `report.html` — مولّد تقارير العملاء.
- `landing.html` — صفحة التعريف.
- `netlify/functions/claude.js` — وكيل المساعد الذكي (متغيّر البيئة: ANTHROPIC_API_KEY).

## التفعيل
1. أنشئ مشروع Supabase وطبّق `motabaq-schema.sql` (خارج هذا المستودع).
2. ضع `SUPABASE_URL` و`SUPABASE_ANON_KEY` في: index.html, auth.html, manage.html, plans.html.
3. للدفع: ضع مفتاح Moyasar العام في plans.html، وفعّل Tokenization للتجديد.
4. النشر تلقائي عبر Netlify المرتبط بهذا المستودع.

## ملاحظة
الوضع الافتراضي «تجريبي» حتى تُضاف مفاتيح Supabase — فلا يتعطّل الموقع الحيّ.
